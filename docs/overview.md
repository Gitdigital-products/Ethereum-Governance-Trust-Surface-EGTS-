# Overview

## What is ETSC Core?
Ethereum Trust Surface Core is a set of schemas, documentation, and conceptual examples for building verifiable trust surfaces on Ethereum.

## Core Concepts
- **Trust Signal** – Any cryptographically signed statement about reputation, endorsement, or challenge.
- **Governance Event** – On‑chain or off‑chain action within a DAO or protocol.
- **Contributor Role** – Permission set assigned to an address or DID.
- **Audit Log** – Immutable record of administrative or governance changes.

## Why a Trust Surface?
Decentralized systems need transparent, machine‑verifiable representations of trust. ETSC provides a common language for different governance tools to interoperate.

## Usage
Consumers implement validation logic using the provided JSON Schemas. No runtime is required – schemas are the contract.
